2.Add changes in stage-

3. Unstaging changes -

git commit -m "m- means message"

5.Pust to centrol repository
git push

6.Create a new branch
git branch branch_name

7.Check current branch
git branch ( "green color" or "*" is active branch )

8. Switch in different branch
git checkout branch_name

9.Merge branch
git checkout master
